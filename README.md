# The Thread: Prospect Run

A GTA2-style top-down sales intelligence game built on [The Thread](https://andredavisme.github.io/the-thread-pdf/) textbook schema.

**Play:** [andredavisme.github.io/the-thread-prospect-run](https://andredavisme.github.io/the-thread-prospect-run/)

## Stack
- Single `index.html` — HTML5 Canvas, vanilla JS
- Supabase backend (`tr_players`, `tr_sessions`)
- GitHub Pages hosting

## Cities
| City | Tier | Mechanic |
|---|---|---|
| Millhaven | I | Baseline — learn the 8-step journey |
| City 2 | II | Hidden 5-min timer + Mr. B rage overlay |
| City 3 | III | Moving NPCs — interact when stopped |
| City 4 | IV | Building-bound pogs + red herring NPCs |
| City 5 | V | Infinity-symbol zones, sparse pogs |

## Mr. Bigglesworth
- **Cigar portrait** — appears in the 📋 Briefing manual (pause-safe, timer-freezing)
